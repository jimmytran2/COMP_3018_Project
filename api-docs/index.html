Class/Course Organizer API Documentation (1.0.0)

Documention for class and course organizer app. Perform CRUD operations on classes, courses, and students

Assignment

Creates a new assignment

Authorizations:
bearerAuth
Request Body schema: application/json
required
id
string

Unique identifier for an assignment

name
string

Title or name of the assignment

description
string

Detailed description of the assignment

subject
string

Subject the assignment is related to

dueDate
string <date-time>

Due date for the assignment (e.g. 2025-12-25)

status
string

Current status of the assignment (i.e., "ongoing", "closed", or "graded")

Responses

/api/v1/assignment

Request samples

  • Payload
Content type
application/json
{
  • "name": "Research Paper",
  • "description": "Write a 10-page paper on climate change.",
  • "subject": "Science",
  • "dueDate": "2025-01-01",
  • "status": "assigned"
}

Response samples

  • 201
Content type
application/json
{
  • "status": "success",
  • "data": {
    • "id": "A123",
    • "name": "Research Paper",
    • "description": "Write a 10-page paper on climate change.",
    • "subject": "Science",
    • "dueDate": "2025-01-01",
    • "status": "assigned"
    },
  • "message": "Assignment created"
}

Student

bearerAuth
Request Body schema: application/json
required
id
string

name
string

email

number [ 0 .. 4.5 ]
